翻译项目使用说明

前言

这是一份大一学生的课设作业，正常应该不对外公开，所以无论使用效果怎么样我们不对他负责，也不会修复其中的bug，本项目全为完成作业

项目结构

- 该项目完全用java和javafx编写，调用吾爱大佬写的ocr识别应用识别文本后传输给ai翻译

- jre25文件夹是jdk25版本的虚拟机运行环境，要求该文件夹要和exe文件放在同一个目录下

- ocr助手是吾爱大佬写的ocr识别工具因为他比较方便并且支持开发HTTP服务所以这里偷偷借用一下

- lib库是javafx打包的库文件

使用方法

- 首先打开Ocr识别工具，并且找到设置打开HTTP服务

- 然后启动软件，如果你本地部署了模型那么可以直接打开设置调节模型的名字，调节完毕可以离线使用

- 如果没有部署则需要借用其他网址的api调用接口，可以问一下豆包或者直接查看官方文档找到他们的接口URL输入到第一行，Api密钥输入第二行，如果没有就保持空白，第三行输入模型名称即可完成设置

- 也可以设置字幕位置然后调节文本出现的地方调节完以后按快捷键可以快速识别当前区域的文本翻译
